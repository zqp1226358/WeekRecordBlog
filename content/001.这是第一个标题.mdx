网站参考了程普的[每周网站模板](https://github.com/weijunext/weekly-boilerplate)，然后采用Vercel去部署。

## 开发记录
网站的主要技术栈是nextjs框架

网站主要配置如下
```csharp
├─ app                # 应用入口
│  ├─ weekly
│  │  └─ [slug]
│  │  │  └─ page.tsx  # 内容详情页
│  ├─ layout.tsx
│  └─ page.tsx        # 网站首页
├─ components         # 组件
│  ├─ ……
├─ config             # 网站配置
│  └─ site.ts
├─ content            # mdx 文件统一放在这里
│  ├─ ……
├─ lib                # 公共工具类
│  ├─ ……
├─ public             # 公共静态资源
│  ├─ ……
├─ styles             # 样式
│  ├─ ……
│  ……
```

后续只需要在content中添加周记内容即可。

## 部署

登录 Vercel，和你的GitHub进行绑定，然后导入项目。
它会自动识别出这是一个nextjs项目，只需要默认无脑点击deploy即可。

Vercel 默认会与 GitHub 仓库集成，实现自动部署：
1. 每次你推送代码到 GitHub 仓库的 main 分支时，Vercel 会自动重新构建和部署你的项目。
2. 你可以在 Vercel 仪表板中查看每次部署的状态和日志。
